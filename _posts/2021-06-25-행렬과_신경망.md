```python
X = np.array([1.0, 0.5])
W1 = np.array([[0.1, 0.3, 0.5],
               [0.2, 0.4, 0.6]]) # 2개의 입력을 받는 3개의 뉴런 - 1층의 구조
               
B1 = np.array( [0.1, 0.2, 0.3] )
```


```python
X.shape, W1.shape, B1.shape
```




    ((2,), (2, 3), (3,))




```python
Z1 = np.dot(X, W1) + B1
Z1
```




    array([0.3, 0.7, 1.1])




```python
def sigmoid(x):
  return 1 / (1+ np.exp(-x))
```


```python
A1 = sigmoid(Z1)
A1
```




    array([0.57444252, 0.66818777, 0.75026011])




```python
W2 = np.array([[0.1, 0.4],
               [0.2, 0.5],
               [0.3, 0.6]])  # 3개의 입력을 받는 2개의 뉴런

B2 = np.array([0.1, 0.2])

Z2 = np.dot(A1, W2) + B2
A2 = sigmoid(Z2)
```
